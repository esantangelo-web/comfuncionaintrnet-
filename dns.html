DNS

El DNS (Domain Name System) és un sistema que tradueix noms de domini (com www.google.com) a adreces IP (com 172.217.3.110) que els ordinadors utilitzen per comunicar-se a través d'Internet. Quan escrius una URL, el teu dispositiu consulta un servidor DNS per obtenir l'adreça IP corresponent al domini. Si el servidor DNS no té la resposta, fa una cadena de consultes fins a obtenir la informació. El DNS fa que sigui més fàcil accedir als llocs web mitjançant noms senzills, en lloc de memoritzar adreces numèriques. Els servidors DNS poden ser recursius (busquen la informació) o autoritaris (tenen la informació definitiva d'un domini). També existeixen diversos tipus d'entrades, com A, MX, o CNAME, que gestionen diferents aspectes de les connexions a Internet. En resum, el DNS és essencial per navegar per Internet de manera fàcil i eficient.
